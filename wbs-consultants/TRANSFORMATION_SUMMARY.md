## 🔧 Customization Guide

### Change Animation Speed

```jsx
transition={{ duration: 0.8 }}  // Slower (default: 0.6)
```

### Change Colors

```jsx
className = "bg-emerald-600"; // Change to any Tailwind color
```

### Add New Service

1. Add to `content.js` services array
2. Add to `detailedContent.js` with full details
3. Component will automatically render it

### Modify Testimonials

Edit `content.js` testimonials array - carousel updates automatically
